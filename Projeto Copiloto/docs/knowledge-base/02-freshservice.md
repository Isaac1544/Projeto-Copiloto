# KB 02 --- FreshService

## O que é

FreshService é uma plataforma ITSM utilizada para gestão de serviços.

## Conceitos relevantes

-   Incidente
-   Problema
-   Mudança
-   Catálogo de Serviços
-   CMDB
-   SLA

## Relação com o projeto

O copiloto deverá apoiar o analista durante a análise do ticket,
sugerindo diagnósticos e rascunhos de resposta.
